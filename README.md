## 🛠️ Common Windows issues and fixes

### Power BI will not open the file

- Make sure you installed Power BI Desktop
- Check that the file ended with `.pbix`
- Move the project folder to a local drive like `C:\Users\YourName\Documents`

### Python says a package is missing

Run this command again in Command Prompt:

```bash
pip install pandas seaborn scipy matplotlib jupyter
```

### Charts do not show data

- Check that the data files still sit in the same folder
- Do not rename the CSV or Excel files
- Keep the full project folder together after extraction

### The ZIP file will not extract

- Download the file again
- Use the built-in Windows extractor
- Try a path with fewer folders in the name
